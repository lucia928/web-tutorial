# 常见问题

## 页面导入样式时，使用link和@import有什么区别？

| 对比项         | link                                                 | @import              |
| -------------- | ---------------------------------------------------- | -------------------- |
| 可加载文件类型 | link是XHTML标签，除了加载CSS外，还可以加载图标、字体 | 只能加载CSS文件      |
| 加载时机       | 页面加载时，link同时加载                             | 页面加载完成后再加载 |
| 权重           | 更高                                                 | 更低                 |
| DOM可控性      | 可以动态插入link标签来加载样式                       | 无法动态插入         |
| 兼容性         | 无兼容性问题                                         | IE5以上              |
